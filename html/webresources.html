  1. Bootstrap Fundamentals
    This site gives an overview of how bootstrap works and a description of many key/fundamental pieces of bootstrap that may come up in ISIT320 assignments
    This site offers a description and gives an example of many key pieces of bootstrap. This site is very helpful to me because this class is my first experience with bootstrap and I think any student with similar experience as me would find this site helpful.

    * * *

  2. Linux Commands Cheat Sheet
    This site provides an organized overview of linux commands and describes the exact function of each command
    This site can be useful to anyone who works in linux environments. The lists of clearly explained commands is easily accessable and can be a handy tool when working in linux terminals. I will use this often in ISIT320

    Rating:
    * * * *

  3. Chrome Devtools Basics
    This site covers how to use chrome developer tools and what to use them for. A few concepts are covered including an explanation of the DOM and some common debugging strategies
    This site can be useful to all developers but developers new to debugging would benefit the most. Throughout school I have had to debug in most classes so this site will be a good resources for me going forward.

    Rating:
    * * * * *

  4. Virtual Machine Instance Docs
    This site is Google clouds documentation giving an overview of Computer Engine instances. The documentation covers fundamentals of VM instances and goes over specific uses and customization options available to the instance owner.
    This site is very useful to all students of ISIT320. First time users of virtual machines would be wise to start in this documentation as well as it is an introductory document. The information on this site can be helpful to all people using VM instances.

    Rating:
    * * *

  5. Basic Git Commands
    This site covers a list of basic Git commands that are commonly used in day to day development. There can be so many commands that developers use daily and keeping a cheat sheet can help the developer quickly access these commands without having to memorize them.
    This site is a very useful tool for me as I use Git commands in school and on my personal projects. This site can be helpful for most developers because it keeps track of many important commands a lot of developers commonly use.

    Rating:
    * * * * *
